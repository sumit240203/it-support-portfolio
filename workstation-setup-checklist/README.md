# Workstation Setup Checklist (New User)

## Before you start
- Confirm user name, department, required apps, and access needs.

## Hardware
- Laptop/desktop, monitor(s), dock, keyboard/mouse
- Power adapter, Ethernet (if needed)

## OS + accounts
- Create/confirm user account
- Set device name (standard naming)
- Enable automatic updates

- Wiâ€‘Fi / Ethernet connectivity
- Browser (Chrome/Edge) + standard extensions
- Email/Calendar (if used)

## Security
- Antivirus/endpoint protection installed and updated
- Screen lock enabled
- Disk encryption (if company policy)

## Final validation
- Reboot
- Verify login works
- Test internet + printing (if required)
- Document: device serial, installed apps, handoff notes
